# RentEase
RentEase is a user-friendly platform for renting properties and finding rental listings. Users can browse various properties, filter by categories like trending, room types, and amenities, and view detailed property information. Built with Node.js, Express, EJS, and MongoDB, RentEase aims to simplify property rental.


Tech Stack:
Frontend: HTML, CSS, EJS (Embedded JavaScript), Bootstrap
Backend: Node.js, Express.js
Database: MongoDB
Authentication: User authentication via Passport.js
